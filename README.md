# Baby Name API

## Team Members

## Project Description
This project is a console-based CRUD application and a RESTful API that provides
historical statistics about U.S. baby names. The data is stored in an SQLite
database and accessed through a FastAPI web service.

The API allows users to query a first name and receive:
- The first year the name appeared
- The most popular year for the name
- The top 10 years ranked by popularity

## Technologies Used
- Python 3
- SQLite
- FastAPI
- Uvicorn
